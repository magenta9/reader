# Use a Non-Activating Playback Overlay

Reader will show a system-level Playback Overlay while speech is preparing or playing, similar to Typeless's voice bar. The overlay defaults near the bottom center of the display containing the current mouse location, can be long-pressed and dragged during the current Playback Session to avoid covering content, does not steal focus from the current app, shows a waveform-style animation, reveals a no-text approximate progress fill only on hover, provides no in-overlay controls, and disappears when the Playback Session ends or fails. Users stop playback with the global Stop Shortcut instead of an overlay close button.
