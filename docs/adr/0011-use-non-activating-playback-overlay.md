# Use a Non-Activating Playback Overlay

Reader will show a system-level Playback Overlay while speech is preparing or playing, similar to Typeless's voice bar. The overlay anchors near the bottom center of the display containing the mouse when the Playback Session starts, does not steal focus or intercept mouse input from the current app, shows a waveform-style animation with a read-only no-text approximate progress fill, provides no in-overlay controls, and briefly distinguishes completion, failure, and explicit stop before disappearing. Users stop playback with the global Stop Shortcut or the Menu Bar Menu instead of an overlay close button.
