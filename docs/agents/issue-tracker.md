# Issue tracker: Linear

Issues and PRDs for this repository live in Linear.

## Location

- Workspace: `devbox-zhang`
- Team: `dev` (`DEV`)
- Project: [`reader`](https://linear.app/devbox-zhang/project/reader-41eb054cdf91)
- External pull requests are not a triage surface.

## Content model

- PRDs are Linear documents attached to the `reader` project.
- Implementation tickets are Linear issues in the `reader` project.
- Issue dependencies use Linear's native blocking relations.
- Triage state uses the labels defined in `triage-labels.md`.
- Completed work uses Linear's native `Done` workflow state.
- Migrated descriptions preserve the original `.scratch/...` path for provenance.

## When a skill says "publish to the issue tracker"

Create or update a PRD document or issue in the Linear `reader` project. Use team `dev` for issues and preserve blocking relations.

## When a skill says "fetch the relevant ticket"

Fetch the referenced Linear issue identifier or project document.

## Local files

`.scratch/` is not the active tracker. Migrated local PRD and issue files were removed after the Linear migration was verified.
